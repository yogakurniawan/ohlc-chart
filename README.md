# OHLC Chart made with React
## Tools

* React 16.8
* Webpack 4
* Babel 7
* Hot Module Replacement
* Styled Components

## How to run

* `git clone git@github.com:https://github.com/yogakurniawan/ohlc-chart.git`
* cd ohlc-chart
* npm install / yarn
* npm start / yarn start
* open `http://localhost:8080/`
